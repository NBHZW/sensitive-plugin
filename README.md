# 项目简介

- 项目名称：敏感内容脱敏组件
- 技术栈：Kotlin
- 项目功能：对于常见的敏感信息进行脱敏，利用自定义注解和反射的方式，不依赖任何第三方框架，实现多样化，高性能的脱敏需求





# 默认脱敏规则

| 默认 / 全替换          | null 和 “” 都返回 “”；非空返回 6 位 replaceChar              | AllReplacement            |
| ---------------------- | ------------------------------------------------------------ | ------------------------- |
| 姓名/微信昵称          | 单字名：姓保留，名替换（如 “张三”→“张 *”）；<br />双字名：姓保留，名首字替换（如 “张三丰”→“张*丰”）；<br />复姓 / 四字名：保留前 1-2 字，后续替换（如 “欧阳锋”→“欧阳*”） | NameReplacement           |
| 身份证号 / 驾驶证号    | 保留前 6 位 + 后 4 位，中间 8 位替换（如 “110101199001011234”→“110101********1234”）；严格脱敏可保留前 3 位 + 后 3 位 | IdCardReplacement         |
| 护照号                 | 保留前 2 位（国别码）+ 后 2 位，中间替换（如 “G12345678”→“G1****678”） | PassportReplacement       |
| 手机号 / 固话          | 手机号：保留前 3 位 + 后 4 位，中间 4 位替换（如 “13800138000”→“138****8000”）；<br />固话：保留区号 + 后 2 位，中间替换（如 “010-12345678”→“010-****5678”） | PhoneReplacement          |
| 电子邮箱               | 保留前缀前 1-2 位 +@及域名，中间替换（如 “zhangsan@163.com”→“zhan***@[163.com](https://163.com/)”“z***@[163.com](https://163.com/)”） | EmailReplacement          |
| QQ 号                  | 保留前 2 位 + 后 2 位，中间替换（如 “12345678”→“12****78”）  | SocialAccountReplacement  |
| 银行卡号 / 信用卡号    | 银行卡号：保留前 6 位 + 后 4 位（如 “6222021234567890”→“622202***\**\*\*\*7890”）；信用卡号：保留前 4 位 + 后 4 位（如 “436742\*\*\*\*\****1234”） | BankCardReplacement       |
| 银行卡 / 支付密码      | 全量替换为星号（如 “123456”→“******”）                       | PasswordReplacement       |
| 账户余额 / 交易金额    | 小额（≤10000）：保留整数位前 1 位（如 “1234.56”→“1***.56”）；<br />大额（>10000）：保留万位及以上（如 “123456.78”→“12****.78”）；<br />超大额（>100 万）：仅保留量级（如 “1234567.89”→“100 万 +”） | AmountReplacement         |
| 信用卡 CVV 码 / 有效期 | CVV 码全脱敏（如 “123”→“***”）；有效期保留月份，隐藏年份（如 “12/25”→“12/**”） | CreditCardInfoReplacement |
| 详细地址               | 保留省 + 市 + 区（县），隐藏街道、门牌号（如 “北京市朝阳区建国路 88 号”→“北京市朝阳区 ****”）；<br />模糊化可保留省 + 市（如 “北京市 *****”） | AddressReplacement        |
| 经纬度                 | 保留前 2 位小数，后续截断或替换（如 “39.9042°N, 116.4074°E”→“39.90°N, 116.40°E”） | LngLatReplacement         |
| 登录账号 / 工号        | 用户名：保留前 2 位，后续替换（如 “user12345”→“us****45”）；工号：保留部门前缀 + 后 2 位（如 “dev00123”→“dev**23”） | AccountReplacement        |
| 密码 / Token           | 全量替换为星号（如 “eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9”→“******************************”） | SecretReplacement         |
| IP 地址（IPv4/IPv6）   | IPv4：保留前两段，后两段脱敏（如 “192.168.1.100”→“192.168.*.*”）；<br />IPv6：保留前 4 段，后段脱敏（如 “2001:0db8:85a3:0000:0000:8a2e:0370:7334”→“2001:0db8:85a3:0000:***\*:\****:***\*:\****”） | IpAddressReplacement      |
| 病历号 / 诊断结果      | 病历号：保留前 3 位 + 后 2 位；<br />诊断结果：隐去关键病症（如 “肺癌晚期”→“** 晚期”） | MedicalInfoReplacement    |
| 营业执照号 / 税号      | 保留前 8 位 + 后 4 位（如 “91110105MA00000000”→“91110105********00”） | EnterpriseInfoReplacement |
